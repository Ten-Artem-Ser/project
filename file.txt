Инструкция для загрузки готового проекта на гит хаб.
Сначало нужно зарегистрироваться

	1 Открыть папку в редакторе
	2 проинициизировать командой "git init"
	3 Создать пустой репозиторий на GitHub
	4 Связать репозитории командой "git remote add origin ссылка на репозиторий".


Инструкция для оправки изменений с лок. реп. в облачный

1 Сами по себе файлы не отслеживаются. Нужно прямо указать на файл как изменение проекта командой "git add file_name" гду file_name - путь к файлу

2 Зафиксировать изменения с помощью git commit -m 'сообщение'
git push -u origin main
